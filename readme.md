## Run Locally

Clone the project

```bash
  git clone https://github.com/MaxBenDuah/clipboard-tailwind-project.git
```

Go to the project directory

```bash
  cd clipboard-tailwind-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run watch
```
